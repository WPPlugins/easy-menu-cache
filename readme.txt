Caches all navigation menus in a single wp_option

== Description ==

This plugin caches all used Menus in a single wp option, saving tons of db queries and speeding up your WordPress installation.

No configurations needed, no reason not to use it!

* Author: [Marco Canestrari](http://www.marcocanestrari.it/)

== Installation ==

1. Upload the files to the /wp-content/plugins/easy-menu-cache/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enjoy!
